## 📥 Getting Started in 3 Steps

### Step 1: Install Python (if not already installed)

**Windows:**
1. Download Python from: https://www.python.org/downloads/
2. Run installer
3. ✅ CHECK "Add Python to PATH" during installation
4. Click Install

**Mac:**
- Python 3 usually pre-installed
- If not: `brew install python3`

**Linux:**
```bash
sudo apt-get update
sudo apt-get install python3 python3-pip
```

### Step 2: Start the Application

**Windows:**
- Double-click `start.bat`

**Mac/Linux:**
- Open Terminal
- Navigate to this folder
- Run: `./start.sh`

You'll see:
```
Backend API will start on: http://localhost:8000
```

### Step 3: Open the User Interface

1. Navigate to the `frontend` folder
2. Double-click `index.html`
3. Your browser opens the application
4. Start using it! 🎉

## 🆘 Need Help?

### Application won't start?
- Check Python is installed: `python3 --version`
- Make sure no other app is using port 8000
- Try restarting your computer

### Can't see my data?
- Your data is in: `excel_schema/LoanManagement_DB.xlsx`
- Open it in Excel to verify (but don't edit!)
- Use the web application to interact with data

### Something looks wrong?
- Press F12 in browser to see console
- Check if backend is running (terminal window)
- Restart the application
